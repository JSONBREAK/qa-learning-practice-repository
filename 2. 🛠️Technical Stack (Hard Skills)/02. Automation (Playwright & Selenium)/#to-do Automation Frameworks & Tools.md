# Automation Frameworks & Tools (แบบย่อ)

## Automation Framework คืออะไร
โครงสร้างที่ช่วยให้การเขียนเทสเป็นระบบ เช่น:
- Test Runner
- Assertions
- Reporting
- Test Data/Config

## Tools ที่ควรรู้ (พื้นฐาน)
- **Playwright**: E2E สำหรับเว็บสมัยใหม่
- **Selenium**: ใช้แพร่หลาย รองรับหลายภาษา
- **Cypress**: ใช้ง่ายสำหรับเว็บ (แต่จำกัดบางเคส)

## องค์ประกอบหลักของ Framework
- **Test Runner** (รันเทส)
- **Assertion Library** (ตรวจผล)
- **Reporting** (สรุปผล)
- **Fixtures/Setup** (เตรียมสภาพแวดล้อม)

## ใช้อะไรตอนไหน (สั้นๆ)
- ต้องการเร็ว + สคริปต์ง่าย → Playwright
- ระบบเก่า/ต้องใช้หลายภาษา → Selenium
